## Description
This is a sample project generated by [Niom](https://github.com/go-niom/niomgo/niom) Framework

## Installation

```bash 
$ go mod tidy
```
## Running the app
```bash
# development mode
$ niom start:dev
# spin up
$ niom start .
```
## Test 

```bash 
# unit tests 
$ go test packageName
```
